# arrowwords.koplugin

An Arrowwords crossword plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Fill the grid with words. Clues appear directly inside the grid cells with an arrow pointing to where the answer goes. Letters at crossing cells must satisfy all words through them. Solve every clue to complete the puzzle.

## Features

- **Bundled puzzles** — several ready-to-play arrowwords grids
- **Cell selection** — tap any answer cell to see its clue and direction
- **Keyboard input** — type letters directly
- **Erase** — clear individual cells
- **Check** — highlights incorrect letters
- **Auto-save** — progress saved and restored on next launch

## Installation

1. Download `arrowwords.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Arrowwords**.

## Controls

| Action | How |
|--------|-----|
| Select a cell | Tap it |
| Enter a letter | Tap the keyboard |
| Erase a letter | Tap **Erase** |
| Check progress | Tap **Check** |
| New puzzle | Tap **New** |
| Show rules | Tap **Rules** |
